# Архитектура проекта роборуки CD4000

## Полная архитектура системы

```

Веб-интерфейс (robot-gripper.html)
│
▼  (команда по USB/Serial)
│
├─── Микроконтроллер Arduino
│    ├── Принимает команды (GRIP / RESET / SET_STEPS)
│    ├── Эмулирует нажатия кнопок ПУСК и СБРОС
│    ├── Управляет DIP-переключателями через 74HC595
│    └── Читает состояние светодиодов через оптроны
│
▼  (сигналы на панель управления)
│
├─── Панель управления (дискретная логика CD4000)
│    ├── DIP-переключатели (8 бит) — задание числа шагов (1–255)
│    ├── Кнопка «ПУСК» — запуск движения
│    ├── Кнопка «СБРОС» — обнуление счётчика
│    ├── RS-триггер (CD4043) — управление мотором
│    ├── Счётчики (CD4510 × 2) — подсчёт импульсов энкодера
│    ├── Компараторы (CD4063 × 2) — сравнение N_зад и Q_тек
│    └── Светодиоды: зелёный (ГОТОВ), красный (АВАРИЯ), жёлтый (ПИТАНИЕ)
│
▼  (сигнал EN_MOTOR)
│
├─── Драйвер мотора (L298N)
│    ├── Получает сигнал EN_MOTOR
│    ├── Питание: +12 В
│    └── Выходы OUT1, OUT2 на мотор
│
▼  (вращение вала)
│
├─── DC-мотор с редуктором
│    ├── Коллекторный, 12 В
│    ├── Редуктор увеличивает крутящий момент
│    └── Вал соединён с катушкой и энкодером
│
├─── Катушка с тросиком
│    ├── Наматывает стальной тросик (0.5–0.8 мм)
│    ├── Тросик проходит через фторопластовые трубки
│    └── Тянет фаланги пальцев (сгибание)
│
├─── Энкодер (инкрементальный, на валу мотора)
│    ├── Генерирует импульсы при вращении
│    ├── Импульсы → счётчик шагов (CD4510 × 2)
│    └── Остановка по заданному числу шагов (DIP-переключатели)
│
▼  (тросик тянет пальцы)
│
├─── Робо-кисть (антропоморфная)
│    ├── Фаланги пальцев (фанера / PLA-пластик)
│    ├── Суставы на металлических штырях (3–4 мм)
│    ├── Пружины на тыльной стороне (разгибание)
│    └── Тросики в Bowden-системе (фторопластовые трубки)
│
├─── Датчики (аварийная остановка)
│    ├── Микрик на кончике пальца (касание предмета)
│    ├── Микрик на пружине троса (натяжение)
│    └── Сигнал /FAULT → мгновенная остановка мотора

```

## Используемый двигатель

- **Тип двигателя:** коллекторный DC-мотор
- **Редуктор:** для снижения оборотов и увеличения крутящего момента
- **Энкодер:** инкрементальный магнитный на валу мотора

## Механизм остановки при сжатии

### 1. Остановка по положению (по градусам) — основной режим

Пользователь выставляет на DIP-переключателях нужное число шагов. Счётчик считает импульсы энкодера. При совпадении с заданным числом схема подаёт сигнал «Стоп» и отключает двигатель.

### 2. Остановка по событию касания — аварийный режим

Микропереключатель на кончике пальца. Если палец касается предмета раньше завершения движения, микрик замыкается, сбрасывает RS-триггер и обесточивает мотор.

### 3. Остановка по натяжению троса — аварийный режим

Концевой выключатель у пружины-предохранителя в разрыве троса. При перегрузке троса пружина растягивается и нажимает на микрик, вызывая немедленную остановку мотора.

## Протокол обмена с Arduino

### Команды (ПК -> Arduino)

| Команда | Байт | Описание |
|---------|------|----------|
| PING | 0x00 | Проверка связи |
| GRIP |
0x01 | Нажать ПУСК |
| RESET | 0x02 | Нажать СБРОС |
| SET_STEPS | 0x03 + байт | Установить шаги (1-255) |

### Ответы (Arduino -> ПК)

| Ответ | Описание |
|-------|----------|
| READY | Готов к работе |
| MOVING | Выполняется движение |
| GRIPPED | Захват выполнен |
| FAULT | Авария |
| RELEASED | Сброс выполнен |

### Подключение Arduino

```

Arduino Pin 2  -> Реле/Транзистор -> Кнопка ПУСК
Arduino Pin 3  -> Реле/Транзистор -> Кнопка СБРОС
Arduino Pin 4  -> 74HC595 DATA
Arduino Pin 5  -> 74HC595 CLOCK
Arduino Pin 6  -> 74HC595 LATCH
Arduino Pin 7  -> Оптрон -> Зеленый светодиод (READY)
Arduino Pin 8  -> Оптрон -> Красный светодиод (FAULT)
Arduino Pin 9  -> Оптрон -> Желтый светодиод (POWER)
Arduino Pin 10 -> Сигнал MATCH с компаратора

```
