# Архитектура проекта роборуки CD4000

## Полная архитектура системы

Веб-интерфейс (index.html) → PHP сервер (index.php) → Arduino → Панель управления CD4000 → Драйвер мотора → DC-мотор → Робо-кисть

## Используемый двигатель
- Тип: коллекторный DC-мотор с редуктором
- Энкодер: инкрементальный магнитный на валу мотора

## Механизм остановки
1. По положению (DIP-переключатели + энкодер)
2. По касанию (микрик на пальце)
3. По натяжению троса (микрик на пружине)

## Протокол обмена с Arduino
| Команда | Байт | Описание |
|---------|------|----------|
| PING | 0x00 | Проверка связи |
| GRIP | 0x01 | Нажать ПУСК |
| RESET | 0x02 | Нажать СБРОС |
| SET_STEPS | 0x03 | Установить шаги |

## API сервера
| URL | Описание |
|-----|----------|
| /status | Получить статус |
| /grip | Сжать руку |
| /release | Разжать руку |
| /set_steps?value=N | Установить шаги |
