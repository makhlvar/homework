# Панель управления робо рукой

**Проект:** сайт, который позволяет изменять и просматривать состояние захвата механической руки (сжата/разжата) через кнопку на странице или командной строкой в URL.

---

## Как начать использование

1. Скачать файл `robot-gripper.html`
2. Открыть его в любом браузере двойным кликом
3. Управлять захватом кнопкой на странице или через поисковую строку

---

## Доступные команды (пишутся в поисковой строке)

`?status=grip` — сжать роборуку

`?status=release` — разжать роборуку

---

## Управление кнопкой на сайте

- Нажать кнопку **✊ СЖАТЬ** — рука сжимается, индикатор становится зелёным
- Нажать кнопку **🖐 РАЗЖАТЬ** — рука разжимается, индикатор становится красным

---

## Текущий статус

Статус отображается на странице:

- 🔴 **РАЗЖАТА** — рука разжата
- 🟢 **СЖАТА** — рука сжата

Состояние сохраняется при обновлении страницы и синхронизируется с URL.
