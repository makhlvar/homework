# Роборука CD4000 с веб-управлением

Сайт, который позволяет изменять и просматривать статус механической руки (сжата/разжата) через кнопку на странице или командной строкой в URL.

## Как начать использование

1. Скачать все файлы проекта в одну папку
2. Запустить PHP сервер: `php -S localhost:8000`
3. Перейти по ссылке: http://localhost:8000
4. Управлять захватом кнопкой на странице или через URL

## Доступные команды (пишутся в поисковой строке)

- `/` — главная страница при запуске
- `/status` — получить текущий статус
- `/grip` — сжать роборуку
- `/release` — разжать роборуку
- `/set_steps?value=150` — установить количество шагов

## Управление кнопкой на сайте

- Нажать кнопку СЖАТЬ (ПУСК) — рука сжимается, индикатор становится зелёным
- Нажать кнопку СБРОС — рука разжимается, индикатор становится красным

## Текущий статус

Статус отображается на странице:
- Зелёный индикатор — ГОТОВА / СЖАТА
- Красный индикатор — РАЗЖАТА
- Жёлтый индикатор — ДВИЖЕНИЕ

Состояние сохраняется в файле status.json и синхронизируется с URL.

## Подключение Arduino (опционально)

| Пин Arduino | Подключение |
|-------------|-------------|
| 2 | Реле кнопки ПУСК |
| 3 | Реле кнопки СБРОС |
| 4 | 74HC595 DATA |
| 5 | 74HC595 CLOCK |
| 6 | 74HC595 LATCH |
| 7 | Оптрон зелёного светодиода |
| 8 | Оптрон красного светодиода |
| 9 | Оптрон жёлтого светодиода |
| 10 | Сигнал MATCH |

## Требования

- PHP 7.4 или новее
- Браузер Chrome, Firefox, Safari или Edge
- Arduino Uno/Nano/Mega (опционально)
