# Роборука CD4000 с веб-управлением

Система управления роботизированной рукой на дискретной логике (CD4000) через веб-интерфейс и Arduino-посредник.

## О проекте

Проект позволяет управлять механической рукой с помощью веб-интерфейса. Arduino выступает посредником между компьютером и панелью управления на микросхемах CD4000, эмулируя нажатия кнопок и управляя DIP-переключателями.

## Как начать использование

1. Скачать файлы `robot-gripper.html` и `robot_arm_controller.ino`
2. Открыть `robot_arm_controller.ino` в Arduino IDE и загрузить на плату Arduino
3. Открыть `robot-gripper.html` в браузере Chrome или Edge
4. Подключить Arduino к компьютеру через USB
5. Нажать кнопку "Подключить Arduino" на сайте и выбрать COM-порт
6. Управлять захватом кнопками на странице

## Доступные команды (URL)

- `?status=grip` — сжать роборуку
- `?status=release` — разжать роборуку

## Управление кнопками на сайте

- Нажать **СЖАТЬ (ПУСК)** — рука сжимается
- Нажать **СБРОС** — рука разжимается

## Текущий статус

Статус отображается на странице и обновляется автоматически:

- Зелёный индикатор — ГОТОВА
- Жёлтый индикатор — ДВИЖЕНИЕ
- Зелёный индикатор — СЖАТА
- Красный индикатор — АВАРИЯ

## Команды протокола (ПК → Arduino)

| Команда | Байт | Описание |
|---------|------|----------|
| PING | 0x00 | Проверка связи |
| GRIP | 0x01 | Нажать ПУСК |
| RESET | 0x02 | Нажать СБРОС |
| SET_STEPS | 0x03 + байт | Установить шаги (1-255) |

## Ответы Arduino

| Ответ | Описание |
|-------|----------|
| READY | Готов к работе |
| MOVING | Выполняется движение |
| GRIPPED | Захват выполнен |
| FAULT | Авария |
| RELEASED | Сброс выполнен |

## Подключение Arduino

| Пин Arduino | Подключение |
|-------------|-------------|
| 2 | Реле/транзистор кнопки ПУСК |
| 3 | Реле/транзистор кнопки СБРОС |
| 4 | 74HC595 DATA |
| 5 | 74HC595 CLOCK |
| 6 | 74HC595 LATCH |
| 7 | Оптрон зелёного светодиода (READY) |
| 8 | Оптрон красного светодиода (FAULT) |
| 9 | Оптрон жёлтого светодиода (POWER) |
| 10 | Сигнал MATCH с компаратора |
